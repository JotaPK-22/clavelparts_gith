# clavelparts
Marketplace especializado en repuestos con compatibilidad asegurada por modelo y versión de tu auto.
Conectamos a los vendedores de repuestos con el clavero (Entusiasta de los autos) que quiere encontrar el repuesto de su auto. El mundo de las autopartes de autos es un tema complejo de abordar por la variedad y cantidad de piezas diferentes que se comercializan incluso para un mismo modelo, muchas veces la misma pieza sirve en un modelo pero por algún cambio menor del fabricante este no sirve para el mismo vehículo pero fabricado unos meses mas tarde, mercado libre no resuelve la problemática de resolverle los problemas a los usuarios con repuestos fáciles de encontrar y con compatibilidad asegurada. Además la cantidad de piezas y productos que muestra mercado libre resulta abrumadora y poco especifica para el vehículo de cada usuario. El usuario necesita una propuesta mas personalizada para su vehículo y sus necesidades. además de ofrecer un ambiente propicio para los fierreros en donde se invite a comprar todo lo necesario para su vehículo e incluso mejorarlo.
